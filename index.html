Политика конфиденциальности для расширения Chrome для сайта OnlyFans.com

Последнее обновление: 24.07.2023

Настоящая Политика конфиденциальности описывает, как расширение Chrome (далее - "расширение") для сайта OnlyFans.com (далее - "сайт") собирает, использует и обрабатывает личную информацию пользователей. Пожалуйста, ознакомьтесь с этой политикой, прежде чем использовать расширение. Продолжая пользоваться расширением, вы соглашаетесь с условиями настоящей Политики конфиденциальности.

1. Собираемая информация: Расширение может собирать следующую информацию: ID профиля пользователя на сайте OnlyFans.com для обеспечения функциональности расширения; Список групп и информацию об участниках, полученную с помощью API сайта OnlyFans.com; Информацию о сообщениях, отправленных с помощью API сайта OnlyFans.com, для целей сохранения в удаленной базе данных на веб-сервере pythonanywhere.com с использованием SQLite.

2. Использование собранной информации: Собранная расширением информация используется исключительно в целях обеспечения функциональности расширения и улучшения опыта пользователей. Данные, сохраненные в удаленной базе данных на сервере pythonanywhere.com, используются только для поддержания работы расширения и не передаются третьим лицам.

3. Хранение данных на сервере pythonanywhere.com: ID профилей пользователей, списки групп и участников, а также данные о сообщениях сохраняются в базе данных SQLite на веб-сервере pythonanywhere.com. Мы предпринимаем все необходимые меры для защиты данных от несанкционированного доступа, изменения, раскрытия или уничтожения. Только авторизованные лица имеют доступ к этим данным, и они обязаны поддерживать конфиденциальность этой информации.

4. Обновления и изменения: Разработчик расширения оставляет за собой право вносить изменения в настоящую Политику конфиденциальности. В случае внесения значительных изменений мы уведомим пользователей, разместив уведомление на странице расширения в Chrome Web Store. Ваше продолжительное использование расширения после таких изменений будет считаться согласием с обновленной Политикой конфиденциальности.
